# ✨ Elementos visuais do projeto

Para deixar o portfólio mais dinâmico e criativo, o site possui diversos efeitos visuais:

🌫 Névoa animada no fundo

✨ Partículas flutuando pela tela

🎮 Menu com estilo de interface de jogo

🐛 Mobs animados atravessando a tela

⚔ Personagem principal flutuando

🖱 Cursor personalizado

Esses detalhes ajudam a criar uma atmosfera semelhante à de um jogo.
